# 英会話の相づち — ネイティブ60人の「返し方」を全部数えた

---

英会話で相手が何か言った。

そのとき、何を返せばいいか分からなくて固まったことはありませんか。

「何か気の利いたことを言わなきゃ」
「文法的に正しい返事を考えなきゃ」

実は、それが間違いの始まりです。

アメリカ人60人の日常会話39,317発話を分析した結果、相手の発言の後に最も多かったのは "Yeah" でした。721回。

会話は「うまい返し」ではなく、「とにかく返す」ことで回っています。

■ 相手の発言の後に来る「最初の1語」

4語以上の発言の後に来る短い応答（1-3語）を全て集計しました。

→ yeah — 721回
→ mhm — 426回
→ okay — 247回
→ oh — 245回
→ no — 178回
→ right — 136回
→ well — 115回
→ oh yeah — 57回
→ really — 28回
→ sure — 25回

上位10個、全てが短い相づちです。

ちなみに "why?"（なぜ？）と聞き返されたのはたった16回。相手は聞き返してきません。ほとんどの場合、相づちを打つだけです。


---

■ 相づちは「手抜き」ではない

言語学の研究では、会話における相づち（Backchannel）は「聞いていますよ」「続けてください」という積極的なシグナルとされています。

Stolcke et al. (2000) の研究によると、英語の電話会話では発話の49%がStatement（発言）で、Statementの後にBackchannel（相づち）が続く確率は26%でした。

つまり、会話で最も大切な能力は「気の利いたことを言う力」ではなく、「相手の話を受け止めて、反射的に返す力」です。

英会話に苦手意識がある人の多くは、「何か言わなきゃ」と思って固まります。でも実際のネイティブ同士の会話では、"Yeah" "Oh" "Right" だけで会話の大きな部分が回っています。


---

【画像: 7つの相づち一覧】

■ 覚えるのは7つだけ

場面ごとに使い分ける必要はありません。どれを返しても会話は続きます。

→ Yeah. — 万能。何にでも使える。一番多い
→ Oh. — 新しい情報を聞いたとき
→ Right. — 同意・納得
→ Okay. — 理解した、次に進んでいい
→ Really? — 驚き・興味。相手が話を続けてくれる
→ I see. — なるほど。理解を示す
→ I know. — わかる。共感。相手と同じ気持ちだと伝える

■ 練習: 反射で返す

相手の発言を読んで、上の7つのどれかで返してみてください。正解はありません。どれを選んでもOKです。「返せた」ことが大事です。

お題1: "I went to this amazing restaurant last weekend."
（先週末すごいレストラン行ったんだ）

お題2: "I've been so busy at work lately."
（最近仕事がすごく忙しくて）

お題3: "My car broke down this morning."
（今朝、車が壊れちゃって）

お題4: "I got a new job."
（新しい仕事が決まったんだ）

お題5: "I haven't been sleeping well."
（最近あんまり眠れてなくて）

お題6: "I think I'm gonna start learning guitar."
（ギター始めようかなと思って）

お題7: "I don't think I can make it tomorrow."
（明日は行けないかも）

お題8: "It was really cold yesterday."
（昨日ほんと寒かったね）

お題9: "I've been watching this show, it's so good."
（最近見てるドラマがすごくいいんだよね）

お題10: "I lost my phone yesterday."
（昨日スマホなくしちゃった）

全部返せましたか？

Yeah. だけでも10問中10問返せます。それでいいんです。

■ 相づち + ひとこと で「会話している感」が出る

慣れてきたら、相づちの後にひとこと足してみてください。

→ "Oh, that's cool." （あー、いいね）
→ "Yeah, I know right." （うん、わかる）
→ "Really? That's too bad." （ほんと？それは大変だね）

相づち + ひとこと。これだけで会話に参加できます。

次の記事では、「自分の言いたいことを英語にする方法」を解説します。主語3つ × 動詞6つで、会話の大半が組み立てられます。
