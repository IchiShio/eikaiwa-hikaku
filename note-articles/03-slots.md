# 英語は「左から3つ埋めるだけ」 — 主語3つ × 動詞6つで会話が回る
# 投稿先: @ichi_eigo / 価格: ¥980
# 画像: セクションごとに1枚必要（別途作成）

---

英語が口から出てこない原因は、文法力でも語彙力でもありません。

「何から言えばいいか分からない」のが原因です。

日本語は動詞が最後に来ます。「昨日 → 駅で → 友達に → 会った」。
英語は動詞が2番目です。「I → saw → my friend」。

この語順の違いが、英語を話せない最大の壁です。

でもコーパスのデータを見ると、この壁は思ったより低いことが分かります。


---

【画像: 3つのスロット図】

■ 英文は3つのスロットで組み立てる

英語の会話文は、左から3つのスロットを埋めるだけで完成します。

→ SLOT 1（誰が）→ SLOT 2（どうする）→ SLOT 3（何を）

"I" → "don't know" → "why."
"You" → "think" → "so."
"I" → "want" → "to go."

これだけです。文法を考える必要はありません。


---

■ SLOT 1: 主語は3つでいい

420万語の会話コーパスから抽出された頻度データによると、代名詞の67.4%を I, you, it の3語が占めています。

→ I — 167,640回（28.4%）— 自分のこと
→ you — 135,217回（22.9%）— 相手のこと
→ it — 128,165回（21.7%）— それ以外

会話では「自分のこと」「相手のこと」「それ以外のこと」の3つしか話題になりません。主語選びに迷う必要はありません。


---

【画像: 6つの動詞カード】

■ SLOT 2: 核の動詞は6つ

アメリカ人60人の日常会話14万語を分析した結果、否定文でも疑問文でも中心にいる動詞は限られています。

→ know — "I don't know" がtrigram 1位（244回）
→ think — "I think" が文頭315回
→ want — "I don't want" が6.4%
→ have — "you have to" が54回
→ get — BNC 48,511回の万能動詞
→ go — BNC 41,630回

この6つの動詞を覚えれば、会話の核はカバーできます。


---

■ 3つの方向: 肯定・否定・疑問

1つの動詞で、don't をつけるか Do you にするかで意味が3方向に分岐します。

know:
→ 肯定: I know. （知ってる）
→ 否定: I don't know. （知らない）
→ 疑問: Do you know? （知ってる？）

think:
→ 肯定: I think so. （そう思う）
→ 否定: I don't think so. （そう思わない）
→ 疑問: Do you think? （そう思う？）

want:
→ 肯定: I want to. （したい）
→ 否定: I don't want to. （したくない）
→ 疑問: Do you want to? （したい？）

have:
→ 肯定: I have to go. （行かなきゃ）
→ 否定: I don't have to. （しなくていい）
→ 疑問: Do you have to? （しなきゃいけないの？）

get:
→ 肯定: I get it. （分かった）
→ 否定: I don't get it. （分からない）
→ 疑問: Do you get it? （分かる？）

go:
→ 肯定: I'll go. （行く）
→ 否定: I'm not going. （行かない）
→ 疑問: Are you going? （行くの？）

6動詞 × 3方向 = 18パターン。ここに SLOT 3（自分の知っている単語）を足すだけで、無限に文が作れます。


---

■ 短い文を and / but / so でつなぐ

短い文を2つつなぐだけで、十分な長さの発言になります。

BNCの会話データで接続詞の頻度はこの順です。

→ and — 90,886回
→ but — 27,233回
→ so — 15,953回

使い方はシンプルです。

→ "I don't know, but I think it's okay."
  分からないけど、大丈夫だと思う。

→ "Yeah, so I want to try it."
  うん、だからやってみたい。

→ "I think it's good, and I want to go again."
  いいと思う、また行きたい。


---

■ 練習: スロットを埋めてみる

日本語のお題を見て、3つのスロットで英語にしてみてください。

お題1: 知らない
→ I / don't know.

お題2: そう思う
→ I / think / so.

お題3: 行きたい
→ I / want / to go.

お題4: 行かなきゃ
→ I / have to / go.

お題5: 分からないけど、大丈夫だと思う
→ I don't know, but I think it's okay.

お題6: やってみたい
→ I / want / to try it.

お題7: そう思わない
→ I / don't think / so.

お題8: 行きたくない
→ I / don't want / to go.

お題9: うん、だから行こう
→ Yeah, so let's go.

お題10: いいと思う、また行きたい
→ I think it's good, and I want to go again.

左から順に埋めるだけ。文法を考える必要はありません。

→ 核の動詞は know / think / want / have / get / go の6つ
→ 肯定・否定・疑問の3方向で18パターン
