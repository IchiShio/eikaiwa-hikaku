# 英語が口から出る人は「主語+動詞」を先に決めている — SVO思考トレーニング30題
# 投稿先: @ichi_eigo / 単体記事 ¥3,980
# 画像: 別途作成

---

ここまでの記事で、3つのことが分かりました。

→ ネイティブの発話の85%は6語以下
→ 会話は相づち（Yeah/Oh/Right）で回っている
→ 英文は「主語→動詞→目的語」の3スロットで組み立てる

データは分かった。パターンも分かった。
でも、いざ会話になると口から出てこない。

なぜか。

「日本語の語順で考えているから」です。


■ 日本語と英語の決定的な違い

日本語は動詞が最後に来ます。

→ 昨日 → 駅で → 友達に → 会った

英語は動詞が2番目に来ます。

→ I → saw → my friend → at the station

日本語で考えると、「昨日、駅で、友達に...」と情報を積み上げて、最後に「会った」が来ます。

でも英語では、最初に「I saw」と言わなければ文が始まりません。

この「主語+動詞を最初に出す」切り替えができないと、頭の中で日本語を英語に変換しようとして固まります。


■ 会話のタイミング: 沈黙の「標準最大値」は約1秒

会話では、相手の発言が終わった後のタイミングが重要です。

Stivers et al. (2009) が世界10言語の日常会話を分析した研究によると、ターン交替（話者が変わるタイミング）の最頻値は0〜200ミリ秒でした。つまり、ネイティブ同士の会話では、相手が話し終わるほぼ同時に次の人が話し始めています。

また、会話分析の研究者 Gail Jefferson (1989) は、英語の会話における沈黙の「標準最大値」は約1秒であると報告しています。1秒を超える沈黙は、会話の中で「不自然な間」として意識されます。

完璧な文を組み立てる時間はありません。

必要なのは、すばやく「主語+動詞」を口から出すことです。残りは後から付け足せばいい。

→ "I think..." と言えば、その後の言葉は考えながら出せる
→ "I don't know..." と言えば、沈黙にはならない
→ "You mean..." と言えば、聞き返しになる

最初の2語が出れば、会話は続きます。


■ なぜ「練習」で口から出るようになるのか

第二言語習得の研究者 DeKeyser は、言語スキルの習得は「宣言的知識（知っている）→ 手続き的知識（できる）→ 自動化（考えずにできる）」の3段階で進むと提唱しています（Skill Acquisition Theory）。

文法ルールを「知っている」だけでは話せません。それを意味のある文脈で繰り返し練習することで「手続き化」し、さらに反復することで「自動化」されます。

この記事のトレーニングは、まさにこの自動化を目的としています。

→ 「知っている」: 英語はSVOの語順である
→ 「できる」: 日本語を見て、主語+動詞を英語で言える
→ 「自動化」: 考えずに主語+動詞が口から出る

ポイントは「声に出す」ことです。頭の中で考えるだけでは手続き化が起きません。声に出して、口の筋肉を使って、実際に「主語+動詞」を発話する。これが自動化への唯一の道です。


■ トレーニングの方法

日本語のお題を見て、英語の「主語+動詞」を声に出してください。

ルール:
→ 日本語を見たら、すぐに「主語+動詞」を口に出す
→ 完璧な文じゃなくていい。主語+動詞だけでOK
→ 目的語や修飾語は後から付け足す
→ 声に出すことが大事。頭の中だけで考えない


■ レベル1: 基本（主語+動詞だけ）

お題1: 知ってる
→ I know.

お題2: 思う
→ I think...

お題3: 分からない
→ I don't know.

お題4: そう思わない
→ I don't think...

お題5: したい
→ I want...

お題6: 行かなきゃ
→ I have to...

お題7: 分かった
→ I get it.

お題8: 行く
→ I'll go.

お題9: 行きたくない
→ I don't want...

お題10: しなくていい
→ You don't have to...


■ レベル2: 主語+動詞+ひとこと

お題11: いいと思う
→ I think it's good.

お題12: 分からないけど...
→ I don't know, but...

お題13: 行きたい
→ I want to go.

お題14: 確認しなきゃ
→ I need to check.

お題15: まだ分からない
→ I don't know yet.

お題16: なぜか分からない
→ I don't know why.

お題17: 大丈夫だと思う
→ I think it's okay.

お題18: やってみたい
→ I want to try it.

お題19: そう思う？
→ Do you think so?

お題20: それ知ってる？
→ Do you know that?


■ レベル3: 相づち+主語+動詞（会話の流れ）

相手の発言を読んで、「相づち + 自分の意見」をすぐに返してください。

お題21: "This restaurant is pretty good."
→ Yeah, I think so.

お題22: "Do you know where the station is?"
→ I don't know, but I think it's near here.

お題23: "Should we leave now?"
→ Yeah, I think we should.

お題24: "I don't think it'll work."
→ Really? I think it's okay.

お題25: "Do you want to come?"
→ Yeah, I want to go.

お題26: "It's getting late."
→ Right, I have to go.

お題27: "Did you check the schedule?"
→ Oh, I don't know yet.

お題28: "What do you think about this?"
→ I think it's good, and I want to try it.

お題29: "I'm not sure about tomorrow."
→ I don't know, but I think it'll be fine.

お題30: "That sounds difficult."
→ Yeah, but I want to try it.


■ なぜ30題で十分なのか

この30題で使っている主語+動詞の組み合わせは、実はたった6パターンです。

→ I think... （意見を言う）
→ I don't know... （分からないと言う）
→ I want... （したいと言う）
→ I have to... （しなきゃと言う）
→ I don't think... （否定する）
→ Do you...? （聞き返す）

コーパスのデータで確認したとおり、この6パターンで会話の大半をカバーできます。

30題を繰り返し声に出すことで、DeKeyser が提唱する「宣言的知識→手続き的知識→自動化」のプロセスが進みます。最初は時間がかかっても、繰り返すほど反応は速くなります。


■ 実践: 1日5分の習慣

→ 朝: レベル1（お題1-10）を声に出す。2分。
→ 昼: レベル2（お題11-20）を声に出す。2分。
→ 夜: レベル3（お題21-30）を声に出す。3分。

全部で7分。これを毎日続けるだけです。

ポイントは「考えずに口から出す」こと。正解を思い出すのではなく、反射で出るまで繰り返してください。

相づちで「参加」し、主語+動詞で「発言」し、and/but/soで「つなぐ」。

この3つができれば、英会話は回ります。
