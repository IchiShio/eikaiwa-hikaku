## 重点施策ロードマップ（部長策定）

### フェーズ1（2026-03〜04）: Tier 1ページの基盤整備

**目標**: Tier 1クリック 10回/月達成

1. `/services/italki/` のtitle改善 → 順位5.8位でクリックゼロを解消（即効性：高）
2. `/` のtitle/metaを「オンライン英会話比較」系キーワードに最適化
3. `/services/toraiz/`・`/services/progrit/` のコンテンツリライト（3,000字以上）
4. 既存Tier 2記事（articles/）から `/` へのCTAリンクを強化

### フェーズ2（2026-05〜06）: Tier 1流入の拡大

**目標**: Tier 1クリック 50回/月達成

1. Tier 1全サービスページのコンテンツを3,000字以上にリライト
2. `/services/` ページ群のFAQページ schema追加
3. Tier 2記事からの内部送客経路を整備

### フェーズ3（2026-07〜09）: スケール
